Suggest a Feature
=================

Feature requests are welcome! Please open an issue with the
``enhancement`` label on the `PyHDC GitHub Issues page
<https://github.com/GNPower/PyHDC/issues>`_.

What makes a good feature request
-----------------------------------

* **Motivation**: why do you need this? What problem does it solve?
* **Proposed API**: a short code sketch showing how the feature would be used
* **Alternatives considered**: what workarounds exist today?
* **Scope**: is this a small addition or a large redesign?

Current development focus
--------------------------

* **Additional encodings**: new encoding schemes from recent VSA literature
  are considered on a case-by-case basis; please link the paper in your request.
* **Performance optimisations**: contributions that improve batched operation
  speed (especially for the PyTorch backend) are very welcome.
